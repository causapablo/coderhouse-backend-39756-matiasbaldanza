API Endpoints

GET /products

Devuelve la lista de todos los productos.

Ejemplos:

GET /products
GET /products?limit=5

Si se agrega el parámetro "limit" con un valor numérico, devuelve la cantidad de productos indicados
Si el parámetro "limit" no se incluye o no es un número válido, el endpoint devolverá todos los productos existentes.
Si el parámetro limit es un número negativo, se devuelven todos los productos menos el número indicado
(En un array de 40 productos, -30 devuelve los primeros 10 productos)

GET /products/:id

Devuelve un producto con el ID suministrado. Si el ID no existe, devuelve un error 404.

Ejemplo:

GET /products/1
Content-Type: application/json

{
    "id": 1,
    "name": "Updated Product",
    "price": 99.99
}
